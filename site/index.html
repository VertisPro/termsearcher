Search Widget - Prototype
SNOMED CT

Provides a google-like autocomplete search widget based on the Problem Diagnosis refset.

Largely based on guidance provided by the IHTSDO SNOMED CT Search and Data Entry Guide. It provides a term search based on:

  • Words entered in any order
  • Phrase Match
  • Identical Term
  • Starts with
  • Ends with

It also provides commands to operate on the input term, a tactic that is very useful to reduce the number of results.

This tool was built from simple open source tools and aims to to demonsrate a widget prototype that can hopefully be developed for SNOMED data entry.

Another feature (under development) is the ability to spell check within the SCT terms domain allowing spelling assistance for difficult terms.

Compatibility:

Works across desktop and mobile browsers
